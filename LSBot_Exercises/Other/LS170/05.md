# LS170
## Problem 5: URL Components

Identify the main components of the following URL: `https://example.com:8080/path/to/resource?key=value#section`

<details>
<summary>Solution:</summary>

- **Scheme:** `https`
- **Host:** `example.com`
- **Port:** `8080`
- **Path:** `/path/to/resource`
- **Query String:** `?key=value`
- **Fragment:** `#section`

</details>
