# PY130 - Closures, Decorators & Partial Functions
## Problem 104: Function Composition with Decorators

Create a decorator called `@compose` that allows you to compose multiple functions together. The decorator should take multiple functions as arguments and apply them in sequence from right to left (like mathematical function composition).

## Examples:

```python
def add_one(x):
    return x + 1

def double(x):
    return x * 2

def square(x):
    return x ** 2

# Create: f(x) = square(double(add_one(x)))
@compose(square, double, add_one)
def process(x):
    return x

print(process(3))  # ((3 + 1) * 2) ** 2 = (4 * 2) ** 2 = 8 ** 2 = 64
```
<details>
<summary>Solution:</summary>

```python

def compose(*operations):
    def deco(func):
        def wrapper(*args, **kwargs):
            total = len(operations)
            result = operations[-1](*args) # perform last operation first
            for idx in range(total - 2, -1, -1): # start from reversed order
                result = operations[idx](result)
            return result
        return wrapper
    return deco

```
