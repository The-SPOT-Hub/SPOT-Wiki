# PY130
## Problem 57: Higher-order Functions

Define what a "higher-order function" is in Python. Provide the two criteria that a function must meet to be classified as such.

<details>
<summary>Solution:</summary>

A higher-order function is a function that operates on other functions. It must meet at least one of these criteria:
1. **Takes one or more functions as arguments**, OR
2. **Returns a function as its result**

</details>
