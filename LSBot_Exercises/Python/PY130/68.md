## Problem 68: Sequential Invocations

Write a function `execute_operations` that takes a list of functions and a starting value. The function should apply each function in the list sequentially to the value.

```python
# Example usage:
operations = [lambda x: x + 1, lambda x: x * 2]
result = execute_operations(operations, 5)
print(result)  # 12 (5 + 1 = 6, then 6 * 2 = 12)
```

<details>
<summary>Solution:</summary>

```python
def execute_operations(functions, starting_value):
    result = starting_value
    for func in functions:
        result = func(result)
    return result

# Example usage:
operations = [lambda x: x + 1, lambda x: x * 2]
result = execute_operations(operations, 5)
print(result)  # 12 (5 + 1 = 6, then 6 * 2 = 12)

# Another example:
operations2 = [
    lambda x: x * 2,    # 5 * 2 = 10
    lambda x: x + 3,    # 10 + 3 = 13
    lambda x: x ** 2    # 13 ** 2 = 169
]
result2 = execute_operations(operations2, 5)
print(result2)  # 169
```
