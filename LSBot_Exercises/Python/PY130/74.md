# PY130
## Problem 74: Logger

Write a higher-order function `logged_call` that takes a function `func` as an argument and returns a new function that logs when the function is invoked and when it's finished executing.

<details>
<summary>Solution:</summary>

```python
def logged_call(func):
    def wrapper(*args, **kwargs):
        print(f"Calling function {func.__name__}")
        result = func(*args, **kwargs)
        print(f"Finished call to {func.__name__}")
        return result
    return wrapper

# Example usage:
def add(x, y):
    return x + y

def multiply(x, y):
    return x * y

# Create logged versions
logged_add = logged_call(add)
logged_multiply = logged_call(multiply)

# Test the functions
result1 = logged_add(3, 5)
print(f"Result: {result1}")
print()

result2 = logged_multiply(4, 6)
print(f"Result: {result2}")

# Output:
# Calling function add
# Finished call to add
# Result: 8
#
# Calling function multiply
# Finished call to multiply
# Result: 24
```
