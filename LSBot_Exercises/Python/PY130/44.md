# PY130
## Problem 44: Squares

Given the list `numbers = [1, 2, 3, 4, 5]`, use `map` and a lambda function to create a new list containing the square of each number.

<details>
<summary>Solution:</summary>

```python
numbers = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, numbers))
# Result: [1, 4, 9, 16, 25]
```

</details>
