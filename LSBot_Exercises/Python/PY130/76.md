## Problem 76: Nested Data Structures 

Consider the following nested dictionary. Use a generator expression and the sum() function to compute and display the total age of the family's male members. The result should be 444.

```python
munsters = {
    'Herman':  {'age': 32,  'gender': 'male'},
    'Lily':    {'age': 30,  'gender': 'female'},
    'Grandpa': {'age': 402, 'gender': 'male'},
    'Eddie':   {'age': 10,  'gender': 'male'},
    'Marilyn': {'age': 23,  'gender': 'female'},
}
```

<details>
<summary>Solution:</summary>

```python
munsters = {
    'Herman':  {'age': 32,  'gender': 'male'},
    'Lily':    {'age': 30,  'gender': 'female'},
    'Grandpa': {'age': 402, 'gender': 'male'},
    'Eddie':   {'age': 10,  'gender': 'male'},
    'Marilyn': {'age': 23,  'gender': 'female'},
}

# Generator expression to get ages of male members
male_ages = (member['age'] for member in munsters.values()
                           if member['gender'] == 'male')

total_male_age = sum(male_ages)
print(total_male_age)  # Output: 444
```
