# PY130
## Problem 65: Filter + Lambda

Given the list `numbers = [1, 2, 3, 4, 5, 6]`, use the built-in `filter` function with a lambda to create a new list containing only the even numbers.

<details>
<summary>Solution:</summary>

```python
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4, 6]

```

</details>
