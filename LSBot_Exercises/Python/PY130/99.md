# PY130 - Closures, Decorators & Partial Functions
## Problem 99: Multiplier Factory

Create a function `make_multiplier(n)` that returns a function that multiplies its argument by `n`.

## Examples:

```python
times_3 = make_multiplier(3)
times_5 = make_multiplier(5)

print(times_3(4))   # 12
print(times_3(10))  # 30
print(times_5(4))   # 20
print(times_5(7))   # 35
```

<details>
<summary>Solution:</summary>

```python
def make_multiplier(n):
    def inner(num):
        return n * num
    return inner

```


</details>
