## Problem 31: Generator Function with Length Filter

Create a generator function that generates the capitalized version of every string in a list of strings whose length is less than 5. Use a single `print` invocation to print all the capitalized strings as a tuple.

<details>
<summary>Solution:</summary>

```python
def capitalize_short_strings(string_list):
    for s in string_list:
        if len(s) < 5:
            yield s.capitalize()

# Example usage:
strings = ['hi', 'hello', 'world', 'python', 'programming', 'code']
print(tuple(capitalize_short_strings(strings)))
```

**Output:**

```
('Hi', 'Code')
```

</details>
