# PY130 - Closures, Decorators & Partial Functions
## Problem 98: Implementing Partial Without functools

Create your own version of `partial` called `my_partial` that works like `functools.partial`. It should take a function and some arguments, and return a new function with those arguments pre-filled.

## Examples:

```python
def multiply(a, b, c):
    return a * b * c

double = my_partial(multiply, 2)
print(double(3, 4))  # 24 (equivalent to multiply(2, 3, 4))

multiply_by_2_and_3 = my_partial(multiply, 2, 3)
print(multiply_by_2_and_3(4))  # 24 (equivalent to multiply(2, 3, 4))
```

<details>
<summary>Solution:</summary>

```python
def my_partial(func, *fixed_args):
    def inner(*more_args):
        return func(*fixed_args, *more_args)
    return inner

def multiply(a, b, c):
    return a * b * c

```


</details>
