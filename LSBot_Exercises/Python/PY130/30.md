## Problem 30: Generator Expression with Length Filter

Create a generator expression that yields capitalized versions of only the strings whose length is at least 5. Then print the resulting values as a tuple using one `print` call.

<details>
<summary>Solution:</summary>

```python
strings = ['hi', 'hello', 'world', 'python', 'programming', 'code']
capitalized_long = (s.capitalize() for s in strings if len(s) >= 5)
print(tuple(capitalized_long))
```

**Output:**

```
('Hello', 'World', 'Python', 'Programming')
```

</details>
