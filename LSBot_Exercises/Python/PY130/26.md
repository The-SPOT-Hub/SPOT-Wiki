## Problem 26: Refactoring for Purity

The `update_record` function below has a side effect. Identify the side effect and rewrite the function to be pure. You may need to change how the function is used.

```python
# Impure function
def update_record(record, key, value):
    record[key] = value
```

<details>
<summary>Solution:</summary>

**Side effect**: The function mutates the input dictionary `record`.

**Pure version:**
```python
def update_record_pure(record, key, value):
    """
    Returns a new dictionary with the updated key-value pair.
    Original dictionary remains unchanged.
    """
    new_record = record.copy()
    new_record[key] = value
    return new_record
```

</details>
