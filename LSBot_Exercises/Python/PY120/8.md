## 8: super() and __init__

What is output and why? What does this demonstrate about `super()`?

```Python
class Animal:
    def __init__(self, name):
        self.name = name

class GoodDog(Animal):
    def __init__(self, color):
        super().__init__(color)
        self.color = color

bruno = GoodDog("brown")
print(bruno)
```
