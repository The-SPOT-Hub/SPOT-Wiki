# PY101
## Problem 10

What does this print and why? What concept is this?

```python
print("Hello\nWorld")
```

<details>
<summary>Solution:</summary>

This demonstrates the escape character `\n`. This will `print` the string `"Hello"` and then create a new line and `print` `"World"`.

</details>
