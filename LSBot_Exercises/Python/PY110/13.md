## Problem 13: Kebabize a String

Modify the kebabize function so that it converts a camel case string into a
kebab case. Non alphabetic characters should be ignored. 

Kebab case separates words with dashes '-'; camel case identifies separate words by upcasing the first character in each new word.

## Examples:

```python
# The below tests should each print True.
print(kebabize('camelsHaveThreeHumps') == 'camels-have-three-humps')
print(kebabize('myCamelHas3Humps') == 'my-camel-has-humps')
```

<details>
<summary>Solution:</summary>

```python
def kebabize(text):
    result = ""

    for c in text:
        if c.isalpha():
            if c.isupper():
                result += f"-{c.lower()}"
            else:
                result += c

    return result
```
