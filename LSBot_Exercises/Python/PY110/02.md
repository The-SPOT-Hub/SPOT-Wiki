## Problem 2: Count of non-touching Pairs

Write a function that takes a list of integers as input and counts the number of
non-touching pairs in the list. A non-touching pair is defined as two equal integers separated by some other integer(s).

## Examples:

```python
non_touching_pairs([1, 2, 5, 6, 5, 2]) # 1
non_touching_pairs([1, 2, 2, 20, 6, 20, 2, 6, 2]) # 3
```

<details>
<summary>Solution:</summary>

```python
def non_touching_pairs(nums):
    count = 0
    n = len(nums)
    for i in range(n):
        for j in range(i + 2, n):  # start from i+2 to ensure not touching
            if nums[i] == nums[j]:
                count += 1
    return count // 2 # return number of pairs
```
