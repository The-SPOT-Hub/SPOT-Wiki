# PY110
## Problem 17: Expanded Form of Number

You will be given a number, and you need to return it as a string in
expanded form.

Note: All numbers will be whole numbers greater than 0.

## Examples:

```python
# The below tests should each print True.
print(expanded_form(12) == '10 + 2')
print(expanded_form(42) == '40 + 2')
print(expanded_form(70304) == '70000 + 300 + 4')
```

<details>
<summary>Solution:</summary>

```python
def expanded_form(num):
    s = str(num)
    result = []
    for index, digit in enumerate(s):
        if digit != '0':
            num_zeros = len(s) - index - 1
            part = f"{digit}{'0' * num_zeros}"
            result.append(part)

    return ' + '.join(result)
```
