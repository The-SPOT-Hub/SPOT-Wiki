## 9: super() and __init__ Parameters

What is output and why? What does this demonstrate about `super()`?

```Python
class Animal:
    def __init__(self):
        pass

class Bear(Animal):
    def __init__(self, color):
        super().__init__()
        self.color = color

bear = Bear("black")
print(bear)
```
