## 12: Collaborator Objects

We raise an error in the code below. Why? What do `kitty` and `bud` represent in relation to our `Person` object?

```Python
class Person:
    def __init__(self, name):
        self.name = name
        self.pets = []

class Pet:
    def jump(self):
        print("I'm Jumping!")

class Cat(Pet):
    pass

class Bulldog(Pet):
    pass

bob = Person("Robert")
kitty = Cat()
bud = Bulldog()
bob.pets.append(kitty)
bob.pets.append(bud)
bob.pets.jump()
```
