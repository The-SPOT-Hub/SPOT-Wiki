## Problem 11: Polymorphism

What is output and why? How does this code demonstrate polymorphism?

```Python
class Animal:
    def eat(self):
        print("I eat.")

class Fish(Animal):
    def eat(self):
        print("I eat plankton.")

class Dog(Animal):
    def eat(self):
        print("I eat kibble.")

def feed_animal(animal):
    animal.eat()

array_of_animals = [Animal(), Fish(), Dog()]
for animal in array_of_animals:
    feed_animal(animal)
```
